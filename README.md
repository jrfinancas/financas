# Finanças JR

Gestão financeira pessoal — app web hospedado via GitHub Pages.

## Acesso

🌐 **https://jrfinancas.github.io/financas/**

## Módulos

- **Dashboard** — Patrimônio líquido, investimentos, budget e despesas históricas
- **Lançamentos** — Normal, Split (dividido), Holerite, Painel IR, Recorrentes
- **Comprovantes** — Leitura por IA, status dedutível/não dedutível, sugestão de pasta

## Dados

Os dados ficam salvos no `localStorage` do navegador de cada dispositivo.
Para sincronizar entre Mac e iPhone, use os botões **Exportar JSON** e **Importar JSON** na aba Histórico.
